# Ad Management Skills and extensions

This is a set of skills, extensions, and scripts that help analyze and manage Google Ads.

## Purpose
The main purpose is to optimize advertising for conversion and spend using available skills, tools, CLI, and API.

## Support Stack
Supporting tools include node.js

**Prefer Ruby** and the officially supported ruby sdk from google https://developers.google.com/google-ads/api/docs/client-libs/ruby

All future work should be written in ruby.

## Near Future
Existing node google-ads-api based api calls should be recreated with the officially supported ruby api.

**examples**: https://github.com/googleads/google-ads-ruby/tree/HEAD/examples
